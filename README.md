### 🛡️ AEGIS — Agentic Execution & Graph-based Intelligence System

> A production-grade, **self-correcting RAG agent** that visualizes its own reasoning as a live, interactive **3D graph**. Implements **Adaptive + Corrective + Self-RAG** with human-in-the-loop approval and full observability.

**What makes it senior-level:** adaptive query routing (vector store / web / direct), document relevance grading with query rewriting, groundedness + relevance self-checks with bounded retry loops, per-node latency metrics, and LangSmith LLM-as-judge evaluation — every answer is cited and auditable.

`LangGraph` · `LangChain` · `LangSmith` · `FastAPI (SSE streaming)` · `FAISS` · `Next.js + Three.js` · `Docker` · `CI/CD`

### 🏎️ F1 Nexus — Full-Stack Formula 1 AI Analytics Platform

> A complete, cloud-deployed AI product: **live race telemetry, ML race predictions, strategy simulation, and LLM-powered post-race analysis** in one polished platform.

**Engineering highlights:** an **XGBoost** model forecasting win/podium probabilities, a **Monte-Carlo** tyre-strategy simulator, an **LLM race analyst** (Llama 3.2) with rule-based fallback, real-time OpenF1 telemetry, and automated session email reminders — Next.js 16 frontend on Vercel, FastAPI + PostgreSQL backend Dockerized on Render.

`Next.js 16` · `React 19` · `FastAPI` · `XGBoost` · `Llama 3.2 / Ollama` · `PostgreSQL` · `FastF1` · `Docker`
